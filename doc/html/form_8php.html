form_dropdown ( $  options,
$  value = null,
$  attributes = null 
)

Generate HTML drop-down (select) field

Parameters:
$options- a key-value pair array for constructing the options for the dropdown list.
mixed$value- value to appear as selected in the list. For single value, use string, for multiple items to be selected, use array.
array$attributes- key-value pair of attributes. Key is attribute name & value is value for that attribute.
Returns:
string - output HTML
 // For example, Following Code:
        form_dropdown(array(
            "male" => "Male",
            "female" => "Female"
        );, 'female');
  // Will Produce:
 <select  name = '*AUTOGENERATED*'><option  name = 'male' value = 'male'>Male</option><option selected = 'selected' name = 'female' value = 'female'>Female</option></select>
